HTML 기본구조

<!doctype html> 문서 유형(html)을 지정하는 선언문
<html lang ="ko"> 웹문서 시작을 알리는 태그 lang속성 "한국어"

<head> 브라우저 정보를 주는 태그
<meta> 문자세트를 비롯한 문서정보
<meta charset="UTF-8"> 문자세트를 비롯한 문서정보
<meta name="viewport" content="width=device-width,initial-scale=1.0"> 모바일 기기 고려하기
<meta http-equiv="X-UA-Compatible" content="IE=edge"> 인터넷익스플로러 브라우저 고려하기
<meta name="keywords" content="html5, 웹표준"> 검색엔진고려하기 해당문서 키워드
<meta name="description" content="html5를 통해 웹 표준 공부하기"> 검색엔진고려하기 해당문서 키워드
<meta name="author" content="Jo a-reum"> 해당문서 소유자 또는 제작자

<body> 실제 브라우저에 표시될 내용

02.텍스트 관련 태그들

텍스트 블록

<hn> 제목표시 n숫자(1~6)
<p> 단락 앞뒤 자동 줄바꿈
<br> 줄 바꾸기
<hr> 수평줄 넣기
<blockquote> 인용문넣기
<pre> 입력하는 그대로 화면표시 *읽어주기 점자표시 건너뜀 주의

텍스트 한 줄

<strong> <b> 굵게 표시하기
<em> <i> 이탤릭체 표시하기
<q> 인용내용 큰따옴표 표시하기
<mark> 형광펜 효과내기
<span> 줄바꿈 없이 영역 묶기
<ruby> 동아시아 글자 표시하기

기타텍스트 관련 태그들
<abbr> 약자 표시. title 속성을 함께 사용가능
<cite> 웹 문서나 포스트에서 참고내용표시
<code> 컴퓨터 인식을 위한 소스코드
<kdd> 키보드 입력이나 음성 명령 같은 사용자 입력 내용
<small> 부가정보처럼 작게 표시해도 되는 텍스트
<sub> 아래첨자
<sup> 위첨자
<s> 취소선
<u> 밑줄
